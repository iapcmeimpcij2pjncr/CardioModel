<h4>To export a static page to GitHub:</h4>

>1. Open the WP Admin Dashboard 
>2. Hover over Simply Static on the left menu, and then click Generate
>    - Note: the settings for Generation should already be set
>3. After Simply Static is done, you can go to Activity Log and press a link to download a ZIP containing the static page
>4. Uncompress the folder and navigate inside of it.
>5. Replace the current files in your local github repository with the files inside of the folder and commit.
>    - Static webpages should first be pushed to the <code>dev</code> branch and then tested, and if they work you can then move the webpage to <code>main</code> branch.
